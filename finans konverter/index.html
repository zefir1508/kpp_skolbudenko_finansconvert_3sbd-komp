Інформація про конвертер

Цей конвертер дозволяє вам швидко та зручно обмінювати валюти на основі актуальних курсів валют, що надаються через API.

Як працює конвертер?

Конвертер використовує API для отримання актуальних курсів валют і дозволяє користувачам обирати дві валюти для конвертації.

Підтримувані валюти:

  • USD - Долар США
  • EUR - Євро
  • UAH - Гривня
  • GBP - Фунт стерлінгів
  • JPY - Єна
  • CAD - Канадський долар
  • AUD - Австралійський долар
  • CHF - Швейцарський франк
  • CNY - Юань
